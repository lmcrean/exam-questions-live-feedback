## Features

### Rich Text Editor
- Basic formatting (bold, italic, lists)
- Markdown support
- Character/word counter
- Auto-save every 5 seconds
- Undo/redo functionality
- Copy/paste support

### Image Upload
- Drag and drop support
- File picker option
- Up to 5 images
- Max 10MB per image
- Preview thumbnails
- Remove/reorder images
- Mobile camera support

### Question Display
- Expandable/collapsible
- Image zoom/lightbox
- Multiple question images
- Clear, readable format

### Mark Scheme
- Optional peek feature
- Collapsible by default
- Image-based mark scheme
- Zoom capability

### Save & Submit
- Auto-save draft
- Manual save option
- Submit with confirmation
- Edit after submission (until review stage)
- Lock at review stage

### Accessibility
- Keyboard navigation
- Screen reader support
- High contrast text
- Large touch targets
- Clear focus indicators

---

## Validation Rules

### Text Requirements
- No minimum word count (flexible)
- Maximum 3000 words
- Must have either text or images (can't submit empty)
- Auto-save prevents data loss

### Image Requirements
- Maximum 5 images
- Supported: JPG, PNG, HEIC
- Maximum 10MB per file
- Optional (not required)
