## Features

### Response Viewing
- **Expand/Collapse**: Individual response cards
- **Preview Mode**: First 150 characters visible
- **Full View**: Complete text and all images
- **Image Gallery**: Multiple images with lightbox
- **Word Count**: Displayed for each response
- **Timestamps**: Relative time (e.g., "5 mins ago")

### Navigation
- **Scroll**: Infinite scroll or load more
- **Search**: Find responses by keyword
- **Sort**: Multiple sorting options
- **Filter**: Filter by criteria
- **Quick View**: Tap to expand inline

### User Experience
- **Your Response**: Clearly marked
- **Visual Hierarchy**: Important info prominent
- **Loading States**: Smooth transitions
- **Responsive**: Works on all devices
- **Accessible**: Keyboard and screen reader support

### Mark Scheme Access
- **Toggle Visibility**: Show/hide mark scheme
- **Always Available**: Can reference during review
- **Image Zoom**: Full-screen mark scheme view
- **Persistent**: Stays visible while scrolling

### Preparation for Voting
- **Read All Responses**: Before voting begins
- **Take Notes**: Mental preparation
- **Compare**: Side-by-side viewing capability
- **Time to Think**: No rush to vote immediately

---

## Interaction Patterns

### Expand/Collapse
1. **Collapsed**: Shows preview, expand button
2. **Click Expand**: Smooth animation to full view
3. **Expanded**: Shows full content, collapse button
4. **Click Collapse**: Smooth animation to preview

### Image Viewing
1. **Thumbnail**: Click to open lightbox
2. **Lightbox**: Full-size image with controls
3. **Navigate**: Arrow keys or buttons
4. **Zoom**: Pinch or +/- buttons
5. **Close**: X button or escape key

### Search
1. **Type Query**: Real-time filtering
2. **Results**: Highlighted matches
3. **Clear**: X button to reset
4. **No Results**: Helpful message

---

## Accessibility

### Screen Readers
- Announce expanded/collapsed state
- Announce number of responses
- Announce image count
- Announce search results

### Keyboard Navigation
- Tab through responses
- Enter to expand/collapse
- Arrow keys in lightbox
- Escape to close modals

### Visual Aids
- High contrast text
- Large touch targets
- Clear focus indicators
- Loading animations
