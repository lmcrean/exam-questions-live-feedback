## Features

### Voting System
- **Ranked Choice**: Select 1st, 2nd, 3rd place
- **Points System**: 1st=3pts, 2nd=2pts, 3rd=1pt
- **Validation**: Must select all 3, no duplicates
- **Own Response**: Blocked from voting
- **Change Votes**: Can modify before submission
- **Confirmation**: Review before final submit

### Response Viewing
- **Expand/Collapse**: View full responses
- **Image Gallery**: View attached images
- **Search**: Find specific responses
- **Sort**: Order by different criteria
- **Quick View**: Preview without expanding

### User Interface
- **Clear Visual Hierarchy**: Rankings prominent
- **Color Coding**: 1st (gold), 2nd (silver), 3rd (bronze)
- **Progress Indicators**: Show voting status
- **Validation Messages**: Clear error feedback
- **Success States**: Confirm submission

### Real-Time Updates
- **Live Counter**: See how many voted
- **Stage Transitions**: Smooth animations
- **Results Loading**: Progress indicators

---

## Validation Rules

### Required Selections
- Must select exactly 3 different responses
- Each position (1st/2nd/3rd) must be filled
- Cannot submit with incomplete votes
- Cannot vote for own response

### Duplicate Prevention
- Same response can't be ranked twice
- Clear error message if attempted
- Visual feedback on unavailable options

### Edit Restrictions
- Can change votes before submission
- Cannot change after final submit
- Lock at results stage

---

## Accessibility

### Screen Readers
- Announce vote selections
- Announce validation errors
- Announce submission success
- Announce results

### Keyboard Navigation
- Tab through responses
- Space/Enter to select vote
- Arrow keys to change selection
- Clear focus indicators

### Visual Aids
- High contrast buttons
- Clear selected state
- Color + text labels (not color alone)
- Loading animations
