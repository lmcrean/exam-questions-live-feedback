## Features

### Navigation Options
1. **Teacher Path**: Login button leads to teacher authentication
2. **Student Path**: Join button leads to session code entry

### Information Display
- App branding and logo
- Tagline explaining core purpose
- Quick overview of workflow
- Clear call-to-action buttons

### Design Notes
- Clean, focused layout
- Two clear paths for different user types
- Educational, approachable tone
- Responsive design for mobile/tablet/desktop
