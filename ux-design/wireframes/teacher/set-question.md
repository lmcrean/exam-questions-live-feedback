## Validation Rules

### Required Fields
- At least one of:
  - Question text (if no images)
  - Question images (if no text)
- Session title is optional but recommended

### Image Validation
- Maximum 5 images
- Supported formats: JPG, PNG, HEIC
- Maximum size: 10MB per image
- Minimum dimensions: 100x100px recommended

### Text Validation
- No maximum length for question text
- Markdown formatting supported
- Auto-save draft every 30 seconds

### Drag and Drop
- Visual feedback when dragging files
- Highlight drop zone
- Multi-file upload support

### Image Preview
- Click thumbnail to view full size
- Lightbox modal for detailed view
- Reorder images by drag and drop
- Remove individual images

### Text Editor
- Markdown support with preview
- Basic formatting toolbar
- Character/word count
- Auto-save functionality

- Mark scheme visibility toggle
- Voting enable/disable
- Peer marking enable/disable
