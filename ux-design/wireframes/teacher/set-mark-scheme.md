## Features

### Upload Capabilities
- Multiple file upload (up to 5 images)
- Drag and drop support
- File type validation (JPG, PNG, PDF)
- Size validation (max 10MB per file)
- Preview before upload

### Image Management
- Reorder by drag and drop
- Remove individual images
- Preview full size
- Zoom and pan
- Download copies

### Visibility Controls
- **Hidden**: Only teacher can see
- **Optional Peek**: Students can choose to view
- **Mandatory**: Students must view before submitting
- View tracking (optional)

### Integration
- Available during session creation
- Editable during submission stage
- Locked after review stage begins
- Displayed in presentation mode
- Accessible in results view

---

## Validation Rules

### File Requirements
- Maximum 5 images
- Supported formats: JPG, PNG, PDF
- Maximum size: 10MB per file
- Minimum dimensions: 100x100px

### Stage Restrictions
- Can add/edit during submission stage
- Cannot modify after review stage starts
- Visibility settings can change anytime
- Students notified of changes in real-time
