## Features

### Session Code Entry
- **6-character alphanumeric code**
- Auto-uppercase conversion
- Auto-submit on complete entry (optional)
- Clear/reset button
- Paste support from clipboard
- Visual feedback on each character

### Username Generation
- **Automatic assignment** (no user input)
- Playful, memorable format
- Emoji + Adjective + Noun + Number
- Unique within session
- Consistent across session
- Fun, student-friendly

### Error Handling
- Invalid code detection
- Session ended notification
- Session full notification (future)
- Network error handling
- Clear error messages
- Easy retry mechanism

### Session Persistence
- Auto-resume on return (same device)
- Session stored in browser
- No password required
- Anonymous participation
- Privacy-focused

### Alternative Join Methods
- Manual code entry (primary)
- QR code scanning (optional)
- Deep link support (future)
- Clipboard paste
