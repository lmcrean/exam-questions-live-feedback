## Features

### View Modes
- **Standard**: Full response with metadata
- **Voting Analysis**: Shows who voted for this response
- **Comparison**: Side-by-side with other responses
- **Mark Scheme**: Split view with marking criteria
- **Timeline**: Edit history and submission flow

### Response Information
- Full text content
- All uploaded images
- Word/character count
- Submission timestamp
- Edit history
- Student username

### Voting Data (Results Stage)
- Total points
- Vote breakdown (1st/2nd/3rd)
- Who voted for this response
- Vote percentage
- Ranking position

### Actions
- Download response (PDF/JSON)
- Print response
- Present on board
- Compare with others
- View with mark scheme
- Navigate between responses

### Image Handling
- Click to zoom/lightbox
- Navigate between images
- Download individual images
- Print with response
